# Mossy

An assembler for the 6502, written in Java.

### Features

- Support for all addressing modes
- Label support

### Pending

- Most directives
  - Namely `.db`/`.dw`, `.include`, and `.macro`
- Named constants
- Arithmetic operations in constant expressions
